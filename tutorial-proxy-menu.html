📌 Introdução

Bem-vindo ao tutorial completo do Proxy Menu! Este é um mod menu avançado para FiveM, com funcionalidades como aimbot, ESP, noclip, teleporte, controle de veículos, armas, admin tools e mais. O acesso e download são **exclusivos para clientes** no servidor oficial do Discord.

Este guia cobre preparação do sistema, requisitos (C++ e Sordum), entrada no servidor, cadastro, injeção e uso. Siga a sequência exata.

🛡️ Aviso Importante – Fevereiro 2026

⚠️ Responsabilidade: O Proxy Menu é um mod menu com funções de cheat para FiveM. O uso viola os Termos de Serviço da Rockstar Games, Cfx.re e FiveM, podendo resultar em ban permanente (HWID, IP, global). FiveGuard e anti-cheat são atualizados constantemente; nenhum método é 100% indetectável. Use apenas em servidores privados com permissão explícita e por sua conta e risco.

🔧 Requisitos

  • Windows 10/11 64 bits
  • Privilégios de administrador
  • FiveM e GTA V legítimo instalados
  • Microsoft Visual C++ Redistributable (2015-2022): Instale x86 e x64 – Download x64 | Download x86
  • Sordum Defender Control (para desativar AV e isolamento de núcleo): Download Oficial
  • Conexão à internet para ativação e injeção
  • Chave de subscrição (fornecida no servidor oficial após compra)
Observação: Instale o Visual C++ antes de tudo. Use o Defender Control para desativar proteções temporariamente (reative após uso).

Passo a Passo: Acesso e Uso

1 Entrar no Servidor Oficial

O download e acesso ao Proxy Menu são **exclusivos para membros** do servidor oficial do Discord. Clique no link abaixo para entrar:

Entrar no Servidor Oficial do Proxy Menu

Importante: Após entrar, vá para o canal de downloads ou abra um ticket de suporte para obter o loader ("Proxader" ou similar) e sua chave de ativação (apenas para clientes pagos/exclusivos). Não baixe de fontes externas.
2 Preparar o Sistema
  1. Instale Microsoft Visual C++ Redistributable (x86 e x64).
  2. Baixe e execute Sordum Defender Control: desative proteção em tempo real e isolamento de núcleo.
  3. Desative qualquer antivírus de terceiros temporariamente.
3 Cadastro e Login no Loader
  1. Execute o loader baixado do servidor oficial como administrador.
  2. Na aba de registro: crie usuário e senha → clique "Register".
  3. Na aba de login: insira suas credenciais, marque "Remember" → clique "Log In".
4 Injeção no FiveM
  1. Abra o FiveM normalmente.
  2. No loader, clique no ícone "Prox" → depois no ícone de download/injeção para carregar o "Prox Executor".
  3. Aguarde a injeção concluir (o menu aparecerá no jogo).
5 Usar o Proxy Menu

Tecla padrão para abrir/fechar o menu: roda do mouse para dentro (pode ser alterada em binds).

  • Aimbot/ESP: Aimbot suave, ESP com nomes/vida/armas, block input.
  • Movimento: Noclip, super run, teleporte (salvo/mapa).
  • Veículos/Armas: Spawn veículos, tunagem infinita, munição sem recuo.
  • Jogadores: TP para jogador, glitch, puxar, chuva de carros.
  • Visual/Admin: Visão noturna/térmica, admins list, FOV changer.

🐛 Troubleshooting

  • Loader não abre/injeta: Verifique instalação de C++, execute como admin, desative AV via Sordum.
  • Erro de chave/injeção: Confirme chave válida no ticket do servidor oficial.
  • Ban detectado: Use conta smurf para testes, evite funções óbvias em servidores públicos.

🔒 Dicas de Segurança

  • Acesse apenas pelo servidor oficial[](https://discord.gg/WNNKFjrYn5).
  • Não compartilhe login/chave (vinculado ao PC).
  • Reative proteções do Windows após uso.
  • Teste em ambiente controlado antes de usar em contas principais.
